Floating login/ register element that sticks to the top of the screen and changes depending on user login state.

== Description ==
A floating element that either displays login/ register or logout/ profile, depending on whether the user is logged in and is a certain user role. Fully customisable colors, links and properties.

== Installation ==
1. Upload `floating-login.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Customise though the floating login panel under settins in your admin area
